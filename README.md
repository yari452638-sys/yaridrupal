
# Aprendizaje Creativo

## Descripción del Proyecto
Este proyecto consiste en la creación de una página web educativa desarrollada en Drupal, orientada a niños de 1ro a 3ro de primaria. El objetivo es ofrecer contenidos dinámicos, creativos e interactivos que ayuden al aprendizaje infantil.

La página incluye:
- Actividades educativas
- Juegos interactivos
- Lectura infantil
- Recursos multimedia
- Espacios de creatividad

---

## Objetivos

### Objetivo General
Desarrollar una plataforma educativa utilizando Drupal para fortalecer el aprendizaje creativo en niños de primaria.

### Objetivos Específicos
- Promover la lectura infantil.
- Incentivar el aprendizaje mediante juegos.
- Integrar recursos visuales y multimedia.
- Crear actividades semanales para los estudiantes.
